# QMBS in spin-1 XY chain
Signatures of many-body scars in bipartite spin-1 XY model and the effects of non-bipartite perturbation on the scarred states, implemented in Julia. 

## Requirements
Julia 1.10.5 or advanced

## Some Points
The main codes use path based storing for efficient operation. Please change the paths accordingly on your system.
